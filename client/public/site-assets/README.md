# Site assets – drop your files here

Copy your images and videos into these folders **using the exact filenames below**. The site will use whatever you put here.

## 1. Landing page videos → `videos/`

| Filename | Where it’s used |
|----------|------------------|
| `hero.mp4` | Full-width background video on the home page hero |
| `landing-buyers.mp4` | Carousel slide “For Buyers” |
| `landing-sellers.mp4` | Carousel slide “Sellers and Agents” |
| `landing-investors.mp4` | Carousel slide “Investors” |

## 2. Signup / login left panel → `signup-left/`

| Filename | Where it’s used |
|----------|------------------|
| `signup.jpg` | Main signup, agency signup, agency invite signup |
| `client-registration.jpg` | Client registration (buyer/tenant/investor/seller) |
| `independent-agent.jpg` | Independent agent registration |
| `login.jpg` | Login page |

Tip: You can use the same image for all by copying it with each filename.

## 3. 9-pic preference grid → `preference-grid/`

| Filename | Where it’s used |
|----------|------------------|
| `1.jpg` … `9.jpg` | “Select your preferences” grid (choose 5 of 9) on client registration |

Use exactly 9 images named `1.jpg`, `2.jpg`, … `9.jpg`.
