## 5. File Upload 413 Fix — Client-Side Splitting

**Commits:** `eea5019`, `39c36b1`

HubSpot and PropData file uploads failed with "413 Content Too Large" because Vercel has a hard 4.5MB request body limit.

### Changes

**`client/src/components/SettingsIntegrations.js`**
- Split large multi-file uploads into separate requests (one file per request) for both HubSpot export and PropData XLSX
- Each file is sent individually, staying under Vercel's 4.5MB limit

**`api/agency/migration/import/hubspot-export.js`**
- Set `module.exports.config = { api: { bodyParser: false, responseLimit: false } }`

**`api/agency/migration/import/propdata-xlsx.js`**
- Set `module.exports.config = { api: { bodyParser: false, responseLimit: false } }`
- Converted from background job + polling to synchronous execution (`runPropdataXlsxImport` / `runPropdataLeanXlsxImport`)

## 10. Enterprise Hierarchy (New Feature)

Added a full enterprise layer above the existing Agency > Branch > Agent structure.

### Schema Changes

**`server/models/User.js`**
- Added `enterpriseId` (ObjectId ref) — links an agency to its parent enterprise
- Added `enterpriseName` (String) — display name of the parent enterprise
- Added `enterpriseStats.agencies[]` — embedded roster of child agencies with `_id`, `name`, `branchCount`, `agentCount`, `status`, `linkedAt`

**`server/models/EnterpriseInvite.js`** (new file)
- Invite model for linking existing agencies or inviting new ones to an enterprise
- Fields: `token`, `enterpriseId`, `enterpriseName`, `agencyEmail`, `agencyId`, `agencyName`, `type` (link/new), `expiresAt`, `accepted`, `declined`

### API Endpoints (8 new serverless functions)

| File | Method | Purpose |
|---|---|---|
| `api/auth/register-enterprise.js` | POST | Register a new enterprise account |
| `api/enterprise/dashboard.js` | GET | Aggregated analytics — agency count, agents, listings, leads, revenue, leaderboard, breakdowns |
| `api/enterprise/agencies.js` | GET | List all linked agencies with performance stats (no PII) |
| `api/enterprise/link-agency.js` | POST | Send invite to an existing agency; creates pending invite if not yet registered |
| `api/enterprise/accept-invite.js` | POST | Agency accepts an enterprise invite |

### PII Protection

**Enterprise CAN see:** Total revenue, properties sold, listing counts, lead counts by type, listings by type, agency leaderboard, agent headcount by tier/branch, lead counts by pipeline status.
